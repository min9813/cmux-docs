# パネル分割時のサイドバータイトルの挙動

## Q: パネルを分割した際、サイドバー（左パネル）のタイトルはどうなるか？

**A:** パネル数によって挙動が変わります（`Workspace.swift:7792-7801`）。

| 条件 | サイドバーのワークスペースタイトル |
|------|----------------------------------|
| パネルが1つだけ & カスタムタイトル未設定 | ターミナルのプロセスタイトル（OSC 0/2）に連動して変わる |
| パネルを分割して2つ以上 | 分割直前のタイトルで固定される（自動更新されなくなる） |
| カスタムタイトルを設定済み | 常にカスタムタイトルが優先 |

各パネル個別のタブタイトル（bonsplit 内のタブバー）は引き続き OSC タイトルに連動して更新されます。影響を受けるのはサイドバーのワークスペース名だけです。

## Q: カスタムタイトルの設定方法は？

**A:** 以下の方法があります。

### キーボードショートカット

| 操作 | デフォルトショートカット |
|------|------------------------|
| ワークスペース名を変更 | `Cmd + Shift + R` |
| タブ名を変更（パネル単位） | `Cmd + R` |

### メニューバー

**View > Rename Workspace...** から変更可能。

### 右クリック

サイドバーのワークスペース行を右クリック → **Rename Workspace...** 。

### cmux.json（設定ファイル）

`~/.config/cmux/cmux.json` のワークスペース定義で `name` を指定:

```json
{
  "workspaces": [
    {
      "name": "Dev Server",
      "cwd": "~/project"
    }
  ]
}
```

### 挙動

- カスタムタイトルを設定すると、OSC タイトル変更やプロセス名の変化に上書きされなくなる（`Workspace.swift:7545-7553`）。
- 空文字列を設定するとカスタムタイトルが解除され、プロセスタイトル連動に戻る。
